# CountBottle (记瓶子) 版本记录

## [V1.4.0] - 2026-06-22

**群组角色与权限模型修复**

### 修复与优化
* 修复平台超管移交群主后，新群主被错误显示为普通成员的问题。
* 统一现任群主、子管理员与普通成员的角色识别，并兼容历史角色数据。
* 群主移交时明确写入新群主身份，原群主同步调整为子管理员。
* 子管理员任免改为群主专属，平台超管保留全局任免与群主移交能力。
* 群组成员列表和 Dashboard 群组卡片新增群主、子管理员身份标记。
* 子管理员保留基础设置、成员认领审核、操作日志和计息规则权限。
* 强制解绑、删除成员、任免子管理员及解散群组保持为群主专属操作。
* 群组成员列表的“未认领”标记调整为中性灰色，避免与群主身份混淆。

## [V1.3.11] - 2026-06-21

**流水与管理日志分页优化**

### 修复与优化
* 成员记账流水改为每批 30 条，并在滚动接近列表底部时自动加载下一批。
* 群组管理日志改为每批 50 条，分类筛选在服务端完成后再分页返回。
* 流水与日志统一使用 Firestore 游标续页，避免重复读取已经展示的数据。
* 累计数量直接使用成员汇总字段，不再为统计扫描全部历史流水。
* 懒加载补充加载中、失败重试和全部加载完成状态。
* 新增流水和管理日志分页所需的 Firestore 复合索引配置。

## [V1.3.10] - 2026-06-20

**数据库读取与接口安全优化**

### 修复与优化
* 页面自动刷新由 5 秒调整为 60 秒，后台标签页暂停刷新，重新聚焦时主动同步。
* 合并群组认领状态查询，并避免同一刷新任务并发重复请求。
* 数据库代理新增集合与字段白名单、查询范围校验和单次返回上限。
* 收紧通用写入接口，仅保留邀请入群和个人排序偏好等必要操作。
* 非本人用户资料仅返回页面展示所需的公开字段。

## [V1.3.9] - 2026-06-19

**计息周期日历语义修复**

### 修复与优化
* 每日、每周、每月和每年计息统一按自然日历推进，不再用固定天数近似月份或年份。
* 每月计息保持管理员设定的日期；遇到短月份时落到月末，后续月份会恢复原设定日期。
* 每年计息正确处理闰年与 2 月 29 日。
* 计息预览与服务端懒计算共用同一套周期规则，补算多期时结果保持一致。

## \[V1.0.0] - 2026-06-17

**🎉 首个正式版本发布**，核心架构与基础功能全面落成！

### 🚀 核心特性 (Features)

* **多端响应式设计**：完美适配移动端与桌面端，深浅色（Dark/Light）模式全局无缝切换。
* **用户鉴权系统**：基于 Firebase Auth 的完整登录流程，支持个人资料（自定义头像与昵称）全平台同步。
* **看板与群组管理 (Dashboard)**：

  * 极简操作台，支持一键创建新群组或输入 20 位 ID 快速加入已有群组。
  * 群组支持自定义计费/计数单位（例如：瓶、元、次）。
  * 支持自动化利息引擎，可配置**单利/复利**以及计息周期（每日/每周/每月/每年）。
* **细粒度权限控制 (RBAC)**：

  * 严格区分 群主 (Creator) > 子管理员 (Admin) > 普通成员 (Member)。
  * 群主专属特权：群组基础设置修改、强制解绑他人账号、彻底删除成员、子管理员任免。
  * 管理员特权：群公告发布、替他人代记账、修改全员备注名。
* **交互的详情页 (Group Details)**：

  * 采用 `framer-motion` 重构的物理回弹动效引擎，支持列表排序（按时间/数量/名字）时的丝滑洗牌重排。
  * 首创“离线代签”模式：管理员可直接为未注册的线下好友创建空白卡片并代为记账，后续好友注册后可随时“认领”属于自己的账本。
  * 账目变动提供顶级反馈：专属 `+1` 浮空粒子动效与类似老虎机的数字滚轮缓冲动画。
  * 支持顶部群组公告栏配置。
* **流水核销追踪 (Records \& Auditing)**：

  * 账目流水精确到人，支持四种类型（增加、核销、强制调平、系统结息）。
  * 完善的负数阻断保护和系统级报错提示。

### 🎨 动效与 UI 设计 (UI \& UX)

* 引入全面 Glassmorphism (毛玻璃) 和深色悬浮卡片美学。
* 基于 TailwindCSS v4 构建全局 CSS 变量体系。
* 引入 Headless UI 配合过渡动画解决 React 渲染销毁时的闪烁/错位问题。
